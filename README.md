# Quantum-Well-Numerical-Methods
Solving the 1D Finite Potential Well using the Finite Difference Method and Transcendental Equations.
This project was aimed at comparing the two numerical methods used to solve Schorindger Equation for finite potential well-the graphical method and finite difference method. Also we've analysed how number of bund staate changes with change in well parameters.
The initial result is calculated  for N=1001, V0=-100eV and Width of well=2e-10
